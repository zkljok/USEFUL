## 下载

### Docker

```
docker pull apachecn0/apachecn-algo-zh
docker run -tid -p <port>:80 apachecn0/apachecn-algo-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install apachecn-algo-zh
apachecn-algo-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-algo-zh
apachecn-algo-zh <port>
# 访问 http://localhost:{port} 查看文档
```
